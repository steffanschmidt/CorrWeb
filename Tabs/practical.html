Microcontrollers and Electronics

I started working with microcontrollers and electronics due to my interest in corrosion from my work at Terma as described in Theoretical Knowledge

Further development of the model required a compact experimental setup, which was able to measure the galvanic voltage difference or potentials precisely and log these over long spans of time.

For this reason I chose to start learning how to work with and program a Teensy, which is a computational powerful Arduino. Read more about the programming of these in Programming.
